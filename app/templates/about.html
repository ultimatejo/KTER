About

This page introduces KTER and kTun.

KTER

KTER began as an umbrella for a range of 'pet projects'. These ranged from excuses to advance my programming skills to tools I would like to be available to me when not using my laptop. The main focus was to create useful engineering tools to aid the tunnel ventilation engineer on their travels.

As the range of tools developed KTER began to translate the tools into web based applications and gathered them together at this website. This remains work in progress and without any warranty. If you wish to use any of the tools here please do so at your own risk. Should you spot any errors or potential for imporvement please let me know.

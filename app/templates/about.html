Kemsing Tunnel Engineering and Research

KTER

KTER began as an umbrella for a range of my 'pet projects'. These ranged from excuses to advance my programming skills to tools I would like to be available to me when not using my laptop. The main focus was to create useful engineering tools to aid the tunnel ventilation engineer on their travels.

As the range of tools developed things got to the point where I could no longer run things without the aid of a dedicated container. This led me into web app programming. This seemed the right approach and KTER became a web presence. This remains work in progress and without any warranty. If you wish to use any of the tools here please do so at your own risk. Should you spot any errors or potential for improvement please let me know.

Not all of the tools that I have developed are available here, although they are descirbed. The reasons for not including them in the tool set vary but if one sounds to be of major interest please get in touch to discuss.
